# Python2 ?? Python3??

Python2와 Python3 버전 중에서 **Python3를 사용해야 한다.**

현재 파이썬 2.7의 버전은 2.7.17이다.

2020년 1월에 2.7 버전의 코드 프리징이 진행되고, 4월에 **2.x의 최종 버전인 2.7.18이 출시**된다. 
**이후로는 더이상 업데이트가 이뤄지지 않는다.**
_Python2 = 죽은 버전_

**Python2와 Python3는 호환이 되지 않는다.**


# Python 설치

Python을 설치 하는 방법은 간단하다.

먼저 [python.org downloads](https://www.python.org/downloads/)에 접속한다.

![](https://images.velog.io/images/gillog/post/246db02b-51bc-457e-8600-5ab40644308c/image.png)


이 중 설치할 버전을 선택하고, Download를 클릭한다.
_나는 3.7.9 버전을 선택했다. 너무 최신버전도 아니고 지원 기간도 길어서..?_

![](https://images.velog.io/images/gillog/post/ac813f9c-cee7-42ec-bf90-e87814df4702/image.png)

자신의 운영체제에 맞는 설치 파일을 선택하여 Download한다.

![](https://images.velog.io/images/gillog/post/9235ab44-1a46-4443-8128-7c4a9f7044fe/image.png)

** !! `Add Python 3.7 to PATH`를 체크해 준다. !! **
Install Now를 선택한다.

![](https://images.velog.io/images/gillog/post/67346955-8736-4266-aeea-12d63ffb9dbf/image.png)


Install이 완료 되었다.

![](https://images.velog.io/images/gillog/post/0a6abe37-e5f9-43aa-85f0-1f7334191a7d/image.png)

Python이 잘 설치 되었는지 확인하기 위해 `IDLE`라는 Python 기본 Shell을 실행한다.

![](https://images.velog.io/images/gillog/post/73f26088-6bba-49cd-8175-8ee700844337/image.png)


Python Shell 에서 `print("Hello World")` 를 실행해 본다.

![](https://images.velog.io/images/gillog/post/b40076b4-f2a8-4a71-b01b-1baa588b60ef/image.png)
